Single Particle Simulation

Simulate the trajectory of a particle in a perfectly circular flow (i.e. solid body rotation), which may represent e.g. an ocean meso-scale eddy.

As an exercise left to the user, directions are provided e.g. to add a convergence / divergence term. For additional documentation e.g. see : 1, 2, 3, 4

    1. setup the software and initialize example
    1. simulate trajectories & plot results
    1. experiment with parameters (user)

1.1 Import Software

using OrdinaryDiffEq, Plots
using IndividualDisplacements, MeshArrays
